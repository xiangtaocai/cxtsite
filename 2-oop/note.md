# oop思想：
    - 面向对象编程技术的关键性观念是它将数据及
    对数据的操作行为放在一起，作为一个相互依存
    不可分割的整体--对象。对于相同类型的对象进
    行分类、抽象后，得出共同的特征而形成了类。面
    向对象编程就是定义这些类。类是描述相同类型的
    对象集合。类定义好之后将作为数据类型用于创建
    类的对象。程序的执行表现为一组对象之间的交互
    通信。对象之间通过公共接口进行通信，从而完成
    系统功能。类中声明的public成员组成了对象的对
    外公共接口。 简单来说就是以功能为解决问题的
    中心.

# 关于self
- self 表示当前对象本身，如果对象调用一个方法，那么该对象会自动传入到当前
　的第一个参数中。
- self并不是关键字，只是接受对象的普通参数。
- 有self成为非绑定类的方法，用对象调用。没有self成为绑定类的方法，
　只能通过类访问。

    - 封装　就是对对象的成员进行访问限制
        - public
            - 没有限制
        - protected
            - 在成员前面加一个下划线，只能内部访问，外部不可以访问？？？？
            　例如from  M import *是不能用的。
        - private
            - 私有是最高级的封装，只能当前类和对象中访问,不能子类和类外访问
            - 在成员前面添加两个下划线即可
                class Person():
                    name = 'klk'
                    #这是公有的
                    __age = 18
                    # 这是私有的成员
            - python中的私有并不是真正的私有，用name mangling技术
            改名的策略，可以用_classname__attribute name 进行访问。
        - public protected private并不是关键字
